# CiberGuardião
Um projeto focado em cibersegurança e proteção digital para o público idoso.

## 🛠️ Tecnologias
- **Backend:** Node.js com Express
- **Frontend:** Vue.js
- **Base de Dados:** MySQL

## 📋 Requisitos do Sistema
O projeto inclui funcionalidades como:
- Conteúdos educativos sobre segurança cibernética;
- Sugestões de tecnologias de prevenção;
- Canal de suporte via Chat.
